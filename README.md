📦 Base64 Encoding: Shifrlangan (bayt ko'rinishidagi) ma'lumotlarni matn sifatida oson nusxalash va yuborish imkoniyati.

🚀 Zero Dependency: Sof Kotlin va Java Crypto API-laridan foydalanilgani sababli dastur o'ta yengil va tezkor.
